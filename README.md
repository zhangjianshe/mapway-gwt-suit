## Build & Install & Deploy

```shell

#如果秩序安装到本地 请窒息感下面命令
git clone https://gitlab.cangling.io/develop/mapway-gwt-suit.git
cd mapway-gwt-suit
mvn clean package install

#发布到 cangling repository
# 如果需要发布到其他仓库 请修改 变量
deploy.sh

```
